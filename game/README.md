# 고덕초 낳기 게임

리듬에 맞춰 올바른 타이밍에 화살표 키를 누르는 리듬 게임입니다!

## 게임 소개

**고덕초 낳기 게임**은 150 BPM 리듬에 맞춰 화면 상단에 나타나는 단어를 보고, 박스가 강조될 때 정확한 입력을 해야 하는 게임입니다.

## 조작 방법

### 키보드
- `↑` (위 화살표): 짜잇호해, 입장해, 들어가, 아무거나해
- `↓` (아래 화살표): 낳아, 퇴장해, 호잇짜해, 아무거나해
- 입력 금지: 고덕초, 낳지마, 들어가지마, `-` (빈칸)

### 화면 버튼
- 왼쪽의 `▲`와 `▼` 버튼 클릭

### 게임 오버 시
- `R` - 게임 다시 시작
- `M` - 메인 메뉴로 이동

## 난이도 시스템

### 라운드 1-2 (쉬움)
- 낳아, 들어가 단어만 등장
- 1~3개 랜덤 배치

### 라운드 3-6 (중간)
- 낳아, 들어가, 낳지마, 들어가지마 등장
- 1~3개 랜덤 배치

### 라운드 7-9 (어려움)
- 모든 단어 등장
- 1~3개 랜덤 배치

### 라운드 10+ (매우 어려움)
- 헷갈리는 단어들 위주
- **무조건 3개 배치**
- 배경색이 연한 빨간색으로 변함

## GitHub Pages 배포 방법

1. GitHub 저장소 생성
2. 파일들을 저장소에 푸시
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/USERNAME/REPOSITORY.git
git push -u origin main
```
3. 저장소 Settings → Pages → Source를 `main` 브랜치로 설정
4. 배포 완료! `https://USERNAME.github.io/REPOSITORY/` 에서 플레이 가능
